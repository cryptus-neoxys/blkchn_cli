# blkchn_cli

Blockchain in a Terminal.

## Running the CLI
**Requirements**
1. Rust
2. Cargo

**Steps to run**
1. ```git clone https://github.com/cryptus-neoxys/blkchn_cli.git```
2. ```cd blkchn_cli```
3. ```cargo build```
4. ```cargo run```

**Create executable**
1. ```cargo build --release```
2. ```cd target/{exec_format}/```
3. ```chmod +x blkchn_cli # or Just run the executable on your platform```
4. ```./blkchn_cli```
